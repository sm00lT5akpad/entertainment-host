About This Project

Introduction

This project showcases a dynamic website for displaying featured movies and series. The home page uses sliders to highlight the top 15 items in each category, while the individual movies and series pages display content in a grid format. Here is a detailed explanation of how the code works:

Home Page

The home page (`home.html`) utilizes sliders to present the featured movies and series. The content is dynamically loaded and limited to the top 15 items from each category:

  • HTML: Contains the structure for the sliders and links to the individual movies and series pages.
  • CSS: Styles the sliders, including hover effects and shadows to enhance visual appeal.
  • JavaScript: Fetches content from the `movies.html` and `series.html` files and appends it to the sliders, limiting the display to 15 items each.

Movies and Series Pages

The movies and series pages (`movies.html` and `series.html`) display content in a grid format. Each item includes an image, title, description, and a link to detailed information:

  • HTML: Contains the structure for the content grid and details sections.
  • CSS: Styles the grid layout, including hover effects and shadows to match the theme.
  • JavaScript: Dynamically loads content from the respective data files (`movies.js` and `series.js`) and displays it in the grid. Provides functionality to show detailed information when an item is clicked.

Data Files

The data files (`movies.js` and `series.js`) contain arrays of objects representing the movies and series. Each object includes properties such as name, picture, description, and video paths for series episodes:

  • Movies Data: Contains an array of movie objects with properties like `name`, `picture`, `description`, and `videoPath`.
  • Series Data: Contains an array of series objects with properties like `name`, `picture`, `description`, and `seasons` (which include episodes).

Styling and Theming

The visual design of the website follows a dark theme with bright accent colors and shadows to enhance the UI. The shadows and hover effects add a modern and dynamic feel to the elements:

  • Colors: A combination of dark backgrounds with bright accent colors (#0093E9, #00B4DB) is used.
  • Shadows: Elements have shadows to create a floating effect and add depth to the design.
  • Hover Effects: Elements scale up and shadows intensify on hover to provide feedback to the user.

Conclusion

This project demonstrates a dynamic and visually appealing way to present movies and series using a combination of HTML, CSS, and JavaScript. The code is designed to be modular and easy to extend, making it a solid foundation for further development.
